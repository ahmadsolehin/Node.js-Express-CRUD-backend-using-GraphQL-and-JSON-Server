# Customerbase GraphQL Server

Node.js/Express CRUD backend using GraphQL and JSON-Server

### Version
1.0.0

## Usage

-Install Dependencies

```bash
$ npm install
```

-Run JSON-Server (Port 3000)

```bash
$ npm run json:server
```

-Run Server (Port 4000)

```bash
$ npm run dev:server
```

-Visit Graphiql IDE

Go to http://localhost:4000/graphql
